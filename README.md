# Random-Password-Generator
A beginner-friendly Python project that generates strong random passwords using uppercase letters, lowercase letters, numbers, and special characters.

## 📖 Introduction
The Random Password Generator is a simple Python project developed using Anaconda that generates strong and secure passwords. Users can specify the desired password length, and the program creates a password containing a combination of uppercase letters, lowercase letters, numbers, and special characters.

This project demonstrates the use of Python libraries, randomization techniques, loops, and user input handling while promoting good cybersecurity practices.

---

## ✨ Features
- Generate secure random passwords
- User-defined password length
- Includes uppercase and lowercase letters
- Includes numbers and special characters
- Simple and beginner-friendly implementation

## 🚀 How to Run the Project

1. Open Anaconda Navigator.
2. Launch Spyder or Jupyter Notebook.
3. Open the `password_generator.py` file.
4. Run the program.
5. Enter the desired password length.
6. The program will generate a secure password.
